=========================================
     MP3Grab v1.0 — Video to MP3 Converter
=========================================

Thanks for using MP3Grab!

MP3Grab converts video files into MP3 audio quickly. It is designed to be
simple: you give it a folder, and it creates an "audio" folder inside it
containing your converted MP3 files.

------------------------------------------------------------
SUPPORTED VIDEO FORMATS
------------------------------------------------------------
• .mp4
• .mkv
• .webm

(If you need more formats added, contact support.)

------------------------------------------------------------
WHAT YOU GET (OUTPUT)
------------------------------------------------------------
When you convert a folder like:

  C:\MyVideos

MP3Grab creates:

  C:\MyVideos\audio\

and saves all MP3s there.

Example:
  C:\MyVideos\movie.mp4
becomes:
  C:\MyVideos\audio\movie.mp3

------------------------------------------------------------
 HOW TO USE (EASIEST METHOD: DRAG & DROP)
------------------------------------------------------------
1) Locate MP3Grab.exe in File Explorer.
2) Drag your VIDEOS FOLDER and drop it on top of MP3Grab.exe.
3) Wait for it to finish converting.
4) When done, open your videos folder and look for: \audio\
   Your MP3 files will be inside.

Notes:
  • MP3Grab processes subfolders automatically (recursive).
  • For the best experience and to see conversion progress, use the
    Command Line Method below.

------------------------------------------------------------
 HOW TO USE (COMMAND LINE METHOD — RECOMMENDED)
------------------------------------------------------------
1) Open Command Prompt:
   Press Windows + R → type: cmd → press Enter

2) Type the following as ONE single line and press Enter:
   "C:\path\to\MP3Grab.exe" "C:\path\to\your\videos\folder"

   Replace the placeholders with your actual paths. For example:
   "C:\Users\John\Desktop\MP3Grab.exe" "C:\Users\John\Videos\Movies"

   TIP: Instead of typing the paths manually, drag MP3Grab.exe
   into the Command Prompt window first, then drag your videos
   folder in after it — the paths fill in automatically!

3) Watch the progress as each file is converted.
4) When finished, look inside your videos folder for:
     \audio\
   Your MP3 files will be there.

------------------------------------------------------------
AUTO-RENAME (NO OVERWRITES)
------------------------------------------------------------
MP3Grab NEVER overwrites existing MP3 files in the "audio" folder.

If an MP3 name already exists, MP3Grab automatically creates a new name:
  intro.mp3
  intro (1).mp3
  intro (2).mp3
...and so on.

This is especially useful when:
• You run MP3Grab multiple times on the same folder, or
• Different subfolders contain videos with the same filename.

------------------------------------------------------------
IMPORTANT NOTES / BEST PRACTICES
------------------------------------------------------------
• Keep MP3Grab.exe in a simple location (example: Desktop) for easy use.
• Do not place MP3Grab.exe inside the folder you are converting.
  (It’s best to keep the EXE separate from your video folder.)
• Conversion speed depends on:
  - your CPU speed
  - video length
  - number of files
• MP3 quality depends on the original audio in the video.

------------------------------------------------------------
TROUBLESHOOTING
------------------------------------------------------------

1) “Nothing happens” when I drag a folder onto the EXE
• Drag-and-drop may not show a progress window on some systems.
• Use the Command Line Method (recommended) to see progress and any
  error messages.

2) Some videos fail to convert
• The video file may be corrupted or encoded in a way that prevents
  decoding.
• Try playing the video normally; if it won’t play, conversion may fail.
• Try converting a different file to confirm MP3Grab is working.

3) I don’t see my MP3 files
• Check inside the folder you converted for:
    \audio\
• Sort by “Date modified” to find the newest files.
